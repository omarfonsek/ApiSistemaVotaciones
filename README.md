# ApiSistemaVotaciones
API RESTful para gestionar un sistema de votaciones. El sistema maneja a los votantes y candidatos, asegurando que cada votante pueda emitir un único voto y a su vez proporciona estadísticas sobre los resultados de la votación

El proyecto fue desarrollado con el framework de Java Springboot, dirigirse a la rama MASTER para bajar los archivos y poder ejecutarlo en alguna IDE, de preferencia intellij IDEA
